Información Proyecto

La generala es un juego de dados tradicional que se juega con cinco dados. El objetivo es conseguir la mayor cantidad de puntos formando combinaciones específicas en un total de 10 o 11 rondas, dependiendo de las variantes. Entre las combinaciones se incluyen números iguales (como tres de un mismo valor), escalera, full, póker y generala (cinco dados con el mismo número).

Cada jugador tiene tres lanzamientos por turno y puede decidir cuáles dados volver a lanzar para intentar formar una mejor combinación. Las reglas principales a respetar son las combinaciones válidas y la cantidad limitada de turnos. La partida finaliza cuando todos los jugadores completan sus turnos, y el que obtiene más puntos es el ganador.
